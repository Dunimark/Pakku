## Options

<snippet id="snippet-options">

`--version`
: Get pakku version

`-y`, `--yes`
: Overwrite every prompt to 'yes' without asking

`--debug`
: Enable additional debug logging

`--working-path=<path>`
: Change the working path of Pakku

`-h`, `--help`
: Show help message and exit
