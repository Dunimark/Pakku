# pakku

## Usage

<snippet id="snippet-cmd">

```
%pakku% [<options>] <command> [<args>]...
```
{prompt="$"}

</snippet>

## Options

<snippet id="snippet-options">

`--version`
: Get pakku version

`-y`, `--yes`
: Overwrite every prompt to 'yes' without asking

`--debug`
: Enable additional debug logging

`-h`, `--help`
: Show help message and exit
{id="option-help"}

</snippet>
