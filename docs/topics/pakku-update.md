# pakku update

Update projects.

## Usage

<snippet id="snippet-cmd">

```
%pakku% update [<options>] [<projects>]...
```
{prompt="$"}

</snippet>

## Options

<snippet id="snippet-options">

`-a`, `--all`
: Update all projects

`-h`, `--help`
: Show help message and exit
{id="option-help"}

</snippet>
